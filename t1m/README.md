## 开发
### 安装
```
npm install
```
### 运行
```
npm run dev
```
浏览自动打开 [http://localhost:9991](http://localhost:9991)

### 多语言
在 [i18n.json] 配置多语言环境，使用"lan"参数匹配语言如：
`https://zsychy.github.io/vue-object-tetris/?lan=en`

`https://zsychy.github.io/vue-object-tetris/?lan=zh`
### 打包编译
```
npm run build
```
